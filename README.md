# FF14高级触发器个人存档

## 监控出售信息：
* 当前v1.5版本
  * A模块监控自ACT打开的所有出售信息，每发现十个出售信息向discord服务器上传信息，合并商品名，重复的不予显示，加算总金额。
  * B模块监控自ACT打开以来所有的传送费用，并且记录第一次传送的时间。用户主动询问后输出到默语频道或是discord服务器
